#  E-commerce Order Processing System

##  Project Description
This project demonstrates the Order Status Lifecycle in an E-commerce system using Java.


##  Order Lifecycle Flow
PENDING → PAYMENT_COMPLETED → SHIPPED → COMPLETED



##  Features
- Order state management using Enum
- Validation rules for status transitions
- Prevents invalid operations (like shipping before payment)



##  Console Output

Initial: PENDING  
After Payment: PAYMENT_COMPLETED  
After Shipping: SHIPPED  
Final: COMPLETED
